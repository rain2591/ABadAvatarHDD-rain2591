		AuroraNXE

	  NXE skin for Aurora 0.7b

		by jveda

Features:

	Xbox Logo with Ripples + Glow Animation
	NXE Font
	NXE Cover-flow replicating MarketPlace layout
	NXE Wallpapers - Loading Bootscreen
	NXE Stage  (use RSS Feed on Button Bar option)
	NXE Sounds (experimental)
	NXE Loading console rings Animation 
	NXE Panels Background

Install:

	Drop files in Aurora Folder
	In Aurora -> (B) View -> Theme
	 Turn off Animation
	 Choose AuroraNXE CoverFlow Layout
	Go To Skin
	 Status when Idle - Hide
	 Choose CleanNXE Skin
        Go to Display
	 Turn off Profile / Disc Panel  (Optional)
	 
       **Turn On RSS Feed for NXE Stage (Optional)**
